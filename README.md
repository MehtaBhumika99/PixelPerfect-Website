# PixelPerfect-Website
Responsive photography website built with HTML and CSS using media queries, developed from a Figma design for Canvas competition.

## Overview

This project is a responsive photography website developed using **HTML** and **CSS** as part of my diploma coursework and a Canvas competition project under the **Responsive Web Design** category.

The website was built based on a provided **Figma design**, with the goal of recreating the layout as accurately as possible. It focuses on practicing responsive design techniques, media queries, and pixel-accurate UI implementation.

## Features

* Fully responsive design using media queries
* Clean photography-themed UI
* Pixel-accurate layout based on Figma design
* Modern and minimal design approach
* Structured sections for portfolio display
* Desktop, tablet, and mobile compatibility

## Technologies Used

* HTML
* CSS
* Media Queries

## Project Type

* Responsive Web Design Project
* Diploma Coursework: Canvas Competition 

## Learning Objectives

This project helped me improve:

* Converting Figma designs into code
* Responsive design using media queries
* Layout structuring with HTML & CSS
* UI/UX accuracy and design consistency
* Frontend development fundamentals
